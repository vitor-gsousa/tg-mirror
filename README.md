# tg-mirror

## Configuração

Copiar .env.example para .env
Editar e inserir valores:

* API_ID
* API_HASH
* DEST_CHAT
* SOURCE_CHATS

## Instalação

```shell
docker compose build
```

## Executar o serviço

```shell
docker compose up -d
```

## Aceder à interface Web

No browser aceder a:
`http://IP_DO_EQUIPAMENTO:8000`
